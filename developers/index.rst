==========
Developers
==========

This page contains useful information for Buster.JS developers/contributors.


.. important::

    You should definitely avoid to have a global installation of Buster.JS while
    developing Buster.JS. Otherwise chances are high that it will be involved when running
    Buster.JS and tests in the development environment.
